# Little Genius

Little Genius is an iOS app designed to help kids learn through fun, interactive activities. It focuses on making learning engaging while giving parents and teachers insight into each child’s progress.

---

## Features

- Interactive mini-games and learning activities
- Progress tracking for learners
- Kid-friendly UI designed for touch
- Built for iPhone and iPad

---

## Requirements

- Xcode 15 or later
- iOS 17.0 or later (update if your target is different)
- Swift and SwiftUI (update if you use UIKit or other frameworks)

---

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/GenukaHettiarachchi/LittleGenius.git
cd "LittleGenius"
```

2. Open the project in Xcode:

```text
Little Genius.xcodeproj
```

3. Select a simulator or a connected device.

4. Build and run from Xcode:

- Product → Run (⌘R)

---

## Project Structure

- `Little Genius/` – App source code
- `Little GeniusTests/` – Unit tests
- `Little GeniusUITests/` – UI tests
- `Little Genius.xcodeproj/` – Xcode project configuration

---

## Running Tests

In Xcode:

- Press ⌘U, or
- Go to **Product → Test**
